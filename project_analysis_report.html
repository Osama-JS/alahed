تاريخ التقرير: 15 نوفمبر 2024

نظرة عامة على المشروع

هذا المشروع هو نظام لإدارة المؤتمرات والمعارض، مبني باستخدام إطار العمل Laravel. يتيح النظام إدارة فعاليات المؤتمرات والمعارض، والمتحدثين، والرعاة، والمعارضين، والجداول الزمنية، والمعارض، والمزيد.

المميزات الرئيسية:

  • نظام إدارة المؤتمرات والفعاليات
  • إدارة المتحدثين والجلسات
  • نظام التسجيل والمشاركين
  • إدارة الرعاة والمعارضين
  • معارض الصور والألبومات
  • لوحة تحكم إدارية متكاملة
  • واجهة مستخدم متجاوبة

المتطلبات التقنية

  • PHP 8.2 أو أحدث
  • Laravel Framework 12.x
  • قاعدة بيانات MySQL / MariaDB
  • Node.js و NPM لإدارة أصول الواجهة الأمامية
  • Composer لإدارة حزم PHP

هيكل المشروع

المجلدات الرئيسية:

  • app/: يحتوي على منطق التطبيق
  • config/: ملفات إعدادات التطبيق
  • database/: يحتوي على ملفات الهجرة والبذور
  • public/: الملفات العامة (CSS, JS, images)
  • resources/: ملفات العرض والموارد
  • routes/
  • storage/: الملفات المرفوعة والكاش
  • tests/: اختبارات التطبيق

كيفية تثبيت وتشغيل المشروع

متطلبات ما قبل التثبيت

  1. تأكد من تثبيت PHP 8.2 أو أحدث
  2. تأكد من تثبيت Composer
  3. تأكد من تثبيت Node.js و NPM
  4. تأكد من تثبيت MySQL / MariaDB

خطوات التثبيت

  1. انسخ المشروع إلى المجلد المطلوب
  2. قم بتشغيل الأمر composer install
  3. قم بنسخ ملف .env.example إلى .env
  4. قم بإنشاء قاعدة بيانات جديدة
  5. قم بتحديث ملف .env بإعدادات قاعدة البيانات
  6. قم بتشغيل الأمر php artisan key:generate
  7. قم بتشغيل الأمر php artisan migrate --seed
  8. قم بتشغيل الأمر npm install
  9. قم بتشغيل الأمر npm run build
  10. قم بتشغيل الخادم باستخدام الأمر php artisan serve
  11. افتح المتصفح على الرابط http://localhost:8000
